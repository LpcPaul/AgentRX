# Legacy Scripts

These scripts were part of the v2.1 architecture. They are preserved for reference but are not part of the current MVP.

| Script | Original purpose |
|---|---|
| `build_index.py` | Rebuild cases/index.json from case files |
| `ci_self_test.py` | Run all validation checks |
| `generate_golden_cases.py` | Generate synthetic seed cases |
| `generate_schema.py` | Generate schema from rules/*.yaml |
| `new_case_id.py` | Generate case IDs |
| `record_outcome.py` | Record case outcomes, auto-verify |
| `retrieve_cases.py` | Deterministic case retrieval |
| `validate_case.py` | Validate case against JSON schema |
